# Changelog

All notable changes to the Lintelos website are documented here.
This project follows [Semantic Versioning](https://semver.org/).

## [1.0.0] - 2026-06-27

Initial release of the Lintelos website design system.

### Design system
- Earth-toned palette built on CSS custom properties (ivory/paper, ink, forest, army,
  avocado, copper/sun) with hairline borders and warm dark sections.
- Typography: Funnel Display for headings (matching the wordmark), Inter for body and UI,
  JetBrains Mono for eyebrow labels, indices, and metadata.
- Components: monospace eyebrow labels, primary/ghost buttons with arrow micro-interaction,
  hairline cards with numbered indices, callout and feature blocks, a numbered process
  timeline, and `--forest`/`--ink` CTA sections with a soft "sun" accent.
- Typographic, no-emoji iconography. Scroll-reveal motion that respects reduced-motion.

### Pages
- Home, About, Consulting, Contact, Products, Archi-Flow Revit add-ins, ROI calculator,
  Privacy Policy, Terms of Service, and Success — plus the shared header and footer.
- Integrations wired and verified: Calendly scheduling, Formspree consulting questionnaire,
  Stripe checkout and billing portal, ROI calculator logic, and tool demo links.
